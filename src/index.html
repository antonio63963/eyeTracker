Eye movement evaluation

The application main function is tracking the speed and coordinates of eye pupils via the mobile device camera.

Download

This is important to know

In the native application built with the OpenCV library integration.

The application is making approximately 1 photo of a patient's face with the 0.5 seconds frequency.

The photos of the patient's face can also be uploaded and processed.

All the computing is processed on the native app side, with no back-end.

Medical application for the eyes

This application is a demo version and is aimed at improving the diagnostic processes of ophthalmic analyzes. This application will support preliminary diagnostics on angular velocity eye pupils measuring.
